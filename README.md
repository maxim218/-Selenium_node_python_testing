# Selenium - пример использования

### Установка Selenium

> Пишем в терминале

sudo pip install selenium

> Заходим в папку по адресу

/usr/local/bin

> Копируем в данную папку файлы

chromedriver

geckodriver

> Теперь файлы находятся по пути

/usr/local/bin/chromedriver

/usr/local/bin/geckodriver
